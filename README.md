### Deploy on Heroku
[![Deploy](https://www.herokucdn.com/deploy/button.svg)](https://heroku.com/deploy?template=https://github.com/Jijinr/Simple-Forward)



Deploy in your vps
```sh
git clone https://github.com/Jijinr/Simple-Forward
cd Telegraph-Uploader-Bot
pip3 install -r requirements.txt
# <Create Variables appropriately>
python3 bot.py
```

### Variables

* `API_HASH` Your API Hash from my.telegram.org
* `API_ID` Your API ID from my.telegram.org
* `BOT_TOKEN` Your bot token from @BotFather
* `TO_CHANNEL` Your Channel ID
